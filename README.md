\---



\## What is AARVAN OS?



AARVAN OS is not a chatbot. It's not an assistant.

It's an \*\*Operating System where AI is the core layer\*\* — not an add-on.



Every interaction goes through an AI brain that understands your intent

and routes it to the right agent to ACT — not just respond.



\---



\## Features



\- 🧠 \*\*AI Brain\*\* — llama3 via Ollama (100% local, offline capable)

\- 🎙️ \*\*Voice Control\*\* — Groq Whisper (95%+ accuracy)

\- 🔊 \*\*Voice Output\*\* — ElevenLabs Vikram (Indian accent)

\- 📁 \*\*File Agent\*\* — manage files by voice

\- 💻 \*\*System Agent\*\* — monitor CPU, RAM, battery

\- 🌐 \*\*Browser Agent\*\* — open any website by voice

\- 📱 \*\*App Agent\*\* — launch any app by voice

\- 🧮 \*\*Calculator Agent\*\* — math by voice

\- 🌤️ \*\*Weather Agent\*\* — live weather (no API key needed)

\- 📝 \*\*Notes Agent\*\* — voice notes

\- 🧠 \*\*Memory Agent\*\* — remembers everything



\---



\## Built On



\- \*\*Base OS\*\*: Ubuntu 26.04 LTS

\- \*\*AI Brain\*\*: Ollama + llama3.2:3b (local)

\- \*\*Voice IN\*\*: Groq Whisper large-v3-turbo

\- \*\*Voice OUT\*\*: ElevenLabs (Vikram voice)

\- \*\*Language\*\*: Python 3.14

